Предсказание смерти на Титанике

Сделано в рамках конкурса kaggle Titanic
В целях изучения языка программирования весь код написан на Java

HTTP Method URI Action
GET /predict Предсказать вероятность по известным данным

Следующие GET параметры обязательны при запросе:

Название Тип Описание
Pclass int Класс билета
Sex string Пол: male / female
Age double Возраст
Siblings_Spouses_Aboard int Кол-во братьев и сестер / супругов на борту Титаника
Parents_Children_Aboard int Кол-во родителей/детей на борту Титаника
Fare double Пассажирский тариф

Примеры ответа

Ответ возвращается в JSON формате.

После запроса на предсказание отмены возвращаются поля:

statusintСтатус работы скрипта (0 = что-то сломалось: 1 = все хорошо).
errorstrОписание ошибки, если она случилась.
valueintПредсказание по выживанию пассажира (1 - выжил или 0 - умер).
